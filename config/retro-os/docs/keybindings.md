# RETRO-OS: Keybindings

## Global

- **Ctrl+Space**: Command palette
- **Cmd+Space**: Raycast (desktop)
- **Cmd+W**: Dismiss current pod
- **Esc**: Return to dashboard
- **Cmd+Q**: Detach from tmux

## Window Navigation

- **Cmd+0**: RETRO-OS window
- **Cmd+1-9**: Other windows (fresh sessions)
- **Cmd+[** / **Cmd+]**: Previous/next window
- **Cmd+T**: New window

## Pane Navigation

- **Alt+H**: Focus left pane
- **Alt+J**: Focus down pane
- **Alt+K**: Focus up pane
- **Alt+L**: Focus right pane

## Tmux Prefix

- **Ctrl+B**: Tmux prefix (for advanced commands)
- **Ctrl+B ?**: Show all keybindings

## Harpoon (Session Bookmarks)

- **Ctrl+H**: Fuzzy jump to bookmarked session
- **Ctrl+Shift+H**: Bookmark current session
